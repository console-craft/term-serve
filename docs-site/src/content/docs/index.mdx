## Built for practical terminal work

<CardGrid stagger>
	<Card title="Real shell sessions" icon="terminal-window">
		Starts an interactive PTY (bash, zsh, fish, etc.) and streams it over WebSocket.
	</Card>
	<Card title="Secure by default" icon="shield-check">
		Binds to `127.0.0.1` by default and auto-enables token auth when exposed beyond localhost.
	</Card>
	<Card title="Phone-ready access" icon="seti:mobile">
		Prints QR codes for LAN and tunnel URLs so you can open the terminal client from your phone quickly.
	</Card>
	<Card title="Cloudflare quick tunnel" icon="networking">
		Use `--tunnel` with an installed `cloudflared` binary for temporary public HTTPS access.
	</Card>
	<Card title="Command mode" icon="seti:terminal">
		Serve one command on connect (`htop`, `btop`, `opencode`, custom scripts) instead of a login shell.
	</Card>
	<Card title="Mobile-capable UI" icon="seti:mobile">
		Includes touch toolbar keys, sticky Ctrl/Alt, and copy mode tuned for scrollback.
	</Card>
	<Card title="Themes and fonts" icon="paintbrush">
		Ships with many terminal themes and configurable font + desktop/mobile font size controls.
	</Card>
	<Card title="Proxy friendly" icon="puzzle">
		Works behind Caddy, nginx, ngrok, and other reverse proxies via forwarded headers.
	</Card>
</CardGrid>
